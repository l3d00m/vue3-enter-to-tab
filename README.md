## Install

NPM:

```bash
npm i --save vue3-enter-to-tab
yarn add vue3-enter-to-tab
```

## Usage

```ts
import { useEnterToTab } from 'vue3-enter-to-tab'

myPackage('hello')
//=> 'hello from my package'
```

**Note**: This directive doesn't work on textarea elements.
